# 🔳 QRlo - QR Code Generator

QRlo is a simple **QR Code Generator website** built using **HTML, CSS, and JavaScript**.  
It allows users to generate QR codes from text or URLs instantly in the browser.

This project is lightweight and works without any backend.

---

## 🚀 Features

- Generate QR Codes instantly
- Works in browser
- Simple user interface
- No installation required
- Fast and lightweight

---

## 🧠 Technologies Used

- HTML
- CSS
- JavaScript

---

## 📁 Project Structure

```
QRlo/
│
└── index.html
```

---

## ▶️ How to Run

1. Download or clone the project

```bash
git clone https://github.com/amitgond1/QRlo.git
```

2. Open the file

```
index.html
```

3. Run in browser

Double click **index.html** or open with Chrome.

---

## 📸 Output

Enter text or a URL and the system will generate a QR code instantly.
